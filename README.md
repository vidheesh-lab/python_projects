# My Python Projects 🚀

Welcome to my repository of beginner-friendly Python scripts! I am currently learning Python fundamentals and building interactive terminal games and utility tools to practice concepts like variables, conditionals (`if/elif/else`), and lists.

## 📁 Included Projects

Here are the projects I have built so far:

*   **`rock-paper-scissors.py`** – A classic text-based game played against the computer using random choice generation.
*   **`banking-terminal.py`** – A simple console application simulating standard ATM functions like deposits and withdrawals.
*   **`bmi-calculator.py`** – A quick health utility tool that calculates Body Mass Index based on user height and weight inputs.
*   **`leap-year-checker.py`** – A logical tool that analyzes any given year and determines if it is a leap year.
*   **`amusement-park-billing-counter.py`** – A fun system that calculates ticket prices based on individual age and height requirements.
*   **`dna-mutation-analyser.py`** & **`cryptoStream-Breaker.py`** – Beginner data exploration scripts evaluating basic sequence analysis and strings.

## 🛠️ Concepts Practiced
*   User Input and Console Printing (`input()` / `print()`)
*   Data Types & Conversions (`int()`, `float()`, `str()`)
*   Conditional Control Flow (`if`, `elif`, `else`)
*   Python Lists and Indexing
*   Importing Core Modules (like `random`)
